# Pixel Perfect

*Это подход к вёрстке сайта, при котором результат вёрстки полностью (вплоть до пикселя) совпадает с дизайнерским макетом.*

Проверить можно в браузере с помощью специальных инструментов. Например, плагина [PerfectPixel](https://www.welldonecode.com/perfectpixel/).

Подробности см. в Доке – https://doka.guide/css/pixel-perfect/
