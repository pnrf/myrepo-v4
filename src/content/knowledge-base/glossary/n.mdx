# Н

**Народно-хозяйственная продукция** – продукция, разрабатываемая и изготовляемая для удовлетворения потребностей народного хозяйства, населения и экспорта.  
<span style={{fontSize: "10px", color: "gray"}}>Источник: ГОСТ 15.201-2000 "Система разработки и постановки продукции на производство. Продукция производственно-технического значения. Порядок разработки и постановки продукции на производство" </span>

---

**Народнохозяйственная продукция (НХП)** – продукция, разрабатываемая и применяемая для удовлетворения потребностей народного хозяйства, населения и экспорта. Продукция, разрабатываемая и изготовляемая для нужд оборы страны, к НХП не относится.  
<span style={{fontSize: "10px", color: "gray"}}>Источник: ГОСТ 15.101-98 "Порядок выполнения научно-исследовательских работ" </span>

---

**Научно-исследовательская работа (НИР)** – комплекс теоретических и (или) экспрериментальных исследований, проводимых с целью получения обоснованных исходных данных, изыскания принципов и путем созданий (модернизации) продукции.  
<span style={{fontSize: "10px", color: "gray"}}>Источник: ГОСТ 15.101-98 "Порядок выполнения научно-исследовательских работ" </span>
