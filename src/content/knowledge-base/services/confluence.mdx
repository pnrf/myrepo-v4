# Confluence

[Confluence](https://www.atlassian.com/ru/software/confluence) – сервис для управления знаниями и совместной работы по проектам. Популярна среди ИТ-разработчиков. Позволяет создать базу знаний по программным продуктам. Часто используется в связке с [Jira](./jira.mdx).

Сервис разрабатывается компанией Atassian. [Ссылка на документацию](https://support.atlassian.com/confluence-cloud/resources/).
