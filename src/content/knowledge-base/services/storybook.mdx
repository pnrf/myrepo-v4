# Storybook 

[Storybook.js](https://storybook.js.org/) – сервис для изолированного создания компонентов пользовательского интерфейса и страниц. Тысячи команд используют его для разработки пользовательского интерфейса, тестирования и документации. У него открытый исходный код и он бесплатный.

Ссылка на документацию – https://storybook.js.org/docs
