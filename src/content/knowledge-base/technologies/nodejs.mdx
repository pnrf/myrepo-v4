# Node.js

[Node.js]('https://ru.wikipedia.org/wiki/Node.js') – серверная платформа, расширяющая возможности языка [JavaScript](./javascript.mdx), в частности для разработки бекэнда веб-приложений. Часто используется в связке с фреймворком [Express.js](./expressjs.mdx). 

[Ссылка на официальный сайт](https://nodejs.org/en).
