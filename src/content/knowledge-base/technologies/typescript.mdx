# Typescript

[Typescript]('https://ru.wikipedia.org/wiki/TypeScript') – язык программирования, расширяющий возможности языка [JavaScript](./javascript.mdx), в частности для статичной типизации, использования полноценных классов ООП и подключения модулей. Компилируется в JavaScript. Часто используется совместно с серверной платформой [Node.js](./nodejs.mdx). 

[Ссылка на официальный сайт](http://www.typescriptlang.org).
