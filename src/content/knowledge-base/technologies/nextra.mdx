# Nextra

***Nextra** – это open-source фреймворк, созданный на основе `Next.js` и `React`, который позволяет создавать статические сайты (SSG) с документацией. Для размещения контента система поддерживает основные возможности `Next.js`, а также разметку `Markdown`.*

*Фреймворк активно поддерживается [сообществом](https://nextra.site/about), включая компанию [Vercel](https://vercel.com), [спонсоров](https://nextra.site/sponsors) и других участников*.

*[Ссылка на официальную документацию Nextra 4.x](https://nextjs.org/docs).*
