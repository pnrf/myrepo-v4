# Markdown

[Markdown](https://ru.wikipedia.org/wiki/Markdown) – облегченный язык разметки для форматирования простого текста. Применяется для машинного преобразования в разметку других языков (HTML, Rich Text и пр.)

[Справочные материалы](https://learn.microsoft.com/ru-ru/contribute/content/markdown-reference).
