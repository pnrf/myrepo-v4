## Основные возможности

### Базовые функции

- **Генерация текста**:  
    — Статьи, письма, посты для соцсетей, сценарии.  
    — Поддержка шаблонов (например, «Напиши письмо в поддержку в формате: приветствие → проблема → решение»).
    
- **Анализ данных**:  
    — Суммаризация длинных документов (например, «Сократи текст до 5 предложений, выделив ключевые решения»).  
    — Классификация текста по темам или тональности (позитивный/негативный отзыв).
    
- **Диалоговый режим**:  
    — Чат-боты для поддержки клиентов, виртуальные ассистенты.  
    — Возможность обучения модели на ваших данных (например, FAQ компании).
    
### Расширенные возможности

- **Fine-tuning**:  
    Настройка модели под специфические задачи (юридические документы, медицинские термины, техническая поддержка).
    
- **Мультимодальность**:  
    Обработка текста в связке с изображениями и таблицами (анализ графиков, извлечение данных из CSV).
    
- **Интеграция с API**:  
    Автоматизация workflows через Яндекс.Облако (например, генерация отчетов каждую пятницу).

## Модели и технические характеристики

### Доступные модели

|Модель|Контекст (токены)|Использование|
|---|---|---|
|**YandexGPT Lite**|4,096|Повседневные задачи, быстрые ответы.|
|**YandexGPT Pro**|10,000|Длинные тексты, аналитика, тонкая настройка.|

### Параметры генерации

- **Температура** (`temperature=0.7`):  
    Чем выше значение (до 1.5), тем креативнее ответы. Для точных данных используйте `temperature=0.2`.
    
- **Длина ответа** (`max_tokens=500`):  
    Ограничивает размер вывода. Для диалогов рекомендуется 100–200 токенов.
    
- **Режимы**:  
    — **Точный**: Минимум креатива, фактологичность.  
    — **Баланс**: Оптимален для большинства задач.  
    — **Креативный**: Подходит для сценариев, стихов, идей.

## Правила написания промптов

### Как формулировать запросы

- **Четкость**:  
    ❌ _«Напиши что-нибудь о маркетинге»_  
		✅ _«Создай план маркетинговой стратегии для запуска фитнес-приложения: 5 шагов, кратко»_.
    
- **Контекст**:  
    Указывайте роль, цель и формат:
    
    ```txt filename="Промпт" copy
    Ты эксперт по питанию. Напиши список из 7 полезных завтраков для спортсменов в формате:  
    Название → Ингредиенты → Время приготовления.  
    ```
    
- **Примеры данных**:  
    Используйте разделители для ввода структуры:
    
    ```txt filename="Промпт" copy
    Проанализируй отзывы ниже и выдели основные жалобы:  
        
    Отзыв 1: «Долгая доставка…»  
    Отзыв 2: «Сломанный товар…»
    ```

### Типичные ошибки

- **Слишком общие запросы**:  
    ❌ _«Объясни квантовую физику»_  
		✅ _«Объясни теорию квантовой запутанности на примере из трех предложений для школьника»_.

- **Игнорирование формата**:  
    ❌ _«Дай список книг»_  
		✅ _«Посоветуй 5 книг в жанре фэнтези с кратким описанием (до 100 символов)»_.

## Ограничения и этика

### Модель не умеет

- Гарантировать абсолютную точность фактов (например, исторических дат).
- Заменять врачей, юристов, финансовых советников и других специалистов.
- Генерировать контент, нарушающий законы РФ (призывы к насилию, дискриминация и пр.).
    
### Рекомендации по этике

- Проверяйте ответы на наличие предвзятости.
- Не используйте модель для создания fake news или deepfakes.

## Интеграция и API

### Подключение через Яндекс.Облако

1. Создайте аккаунт в Яндекс.Облаке.
2. Получите API-ключ в разделе «ИИ-сервисы».
3. Пример запроса на Python:

```python filename="Python" copy
from yandex_gpt import GPTClient  
client = GPTClient(api_key="ВАШ_КЛЮЧ")  
response = client.generate(  
    prompt="Сгенерируй идеи для новогоднего корпоратива:",  
    temperature=0.9,  
    max_tokens=300  
)  
print(response.text)  
```

### Тарифы

- **Бесплатный тариф**: 1000 запросов/месяц (модель Lite).
- **Pro-версия**: От 5000 ₽/месяц (поддержка длинных контекстов, приоритетная очередь).

## Частые вопросы (FAQ)

**Вопрос**: Почему модель иногда «галлюцинирует»?  
**Ответ**: YandexGPT обучается на общих данных и не имеет доступа к актуальной информации после 2023 г. Всегда проверяйте критичные факты.

**Вопрос**: Как обрабатывать ошибки «Слишком длинный запрос»?  
**Ответ**: Укоротите текст или перейдите на модель YandexGPT-Pro с поддержкой 10k токенов.
