# О проекте

В 2023–2024 годах я работал в компании, которая разрабатывала программные решения для медицинской отрасли:

- **ИИ-сервисы** для диагностики патологий на медицинских снимках (маммография, рентгенография, КТ/НДКТ) по различным нозологиям;

- **разметчик изображений** для создания верифицированных наборов данных (дата-сетов), используемых в обучении нейронных сетей;

- **программный комплекс** для автоматизации процессов разработки ИИ-сервисов и их интеграции в медицинскую информационную систему.

Компания принимала активное участие в [государственном эксперименте](https://mosmed.ai/ai/) по использованию инновационных технологий в области компьютерного зрения для анализа медицинских изображений и дальнейшего применения этих технологий в системе здравоохранения.

Компания успешно получала гранты от [Центра диагностики и телемедицины](https://telemedai.ru/) Департамента здравоохранения города Москвы. Кстати, именно это учреждение разработало ГОСТы 59 серии, касающиеся применения ИИ в медицине.
