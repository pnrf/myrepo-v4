# Мой репозиторий

### Общая информация
Проект **"Мой репозиторий"** – это сайт-портфолио технического писателя. Мой личный сайт для поиска работы. "Песочница" для моего развития навыков программирования, составления текстовых описаний и графических схем. 

Проект содержит портфолио с документацией, базу знаний и блог. Доступен для широкой аудитории по адресу: [https://www.myrepo.ru](https://www.myrepo.ru).

### Технологический стек
Сайт создан на фреймворке [Nextra 4.x](https://nextra.site). Программный код размещен в публичном репозитории на [GitHub](https://github.com/pnrf). Хостинг и SSL-протокол предоставлены платформой [Vercel](https://vercel.com). Доменное имя зарегистрировано в [Ру-центре](https://nic.ru). PDF-документы размещены на Google Drive.

* Проект не использует сервисы Яндекса и Google (пока)
