# ИИ-сервисы

В 2023-2024 годах я работал в компании, которая создавала программные решения для медицинской отрасли. 

## О компании

Компания-разработчик сервисов искусственного интеллекта (ИИ-сервисов). Участник "Эксперимента по использованию инновационных технологий в области компьютерного зрения для анализа медицинских изображений", проводимого Центром диагностики и телемедицины Департамента здравоохранения Москвы.

## Ключевые продукты

- Специализированные ИИ-сервисы для лучевой диагностики;
- ПО для разметки медицинских изображений и формирования дата-сетов;
- Программный комплекс по автоматизации процесса разработки ИИ-сервисов.
