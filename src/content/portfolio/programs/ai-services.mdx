# ИИ-сервисы для медицины

В 2023-2024 годах я работал техническим писателем в компании, которая занималась разработкой сервисов искусственного интеллекта (ИИ-сервисов) для медицины. 

Разработка – дело трудоемкое и затратное. Чтобы принять решение, руководству компании требовалось получить одобрение инвесторов. В мои обязанности входило собрать информацию и заполнить бриф с предварительным описанием планируемого сервиса. Привожу пример подобного описания.

В последствии это описание легло в основу конкретных документов:

 - Руководство по эксплуатации
 - Описание функциональных характеристик
 - Технические условия
 - Техническое задание
