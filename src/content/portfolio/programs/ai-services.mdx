# ИИ-сервисы для медицины

В 2023-2024 годах я работал в компании, которая разрабатывала сервисы искусственного интеллекта (ИИ-сервисы) для медицины и сопутствующее программное обеспечение. 

## О компании

Компания-разработчик сервисов искусственного интеллекта (ИИ-сервисов) для медицины. Участник *"Эксперимента по использованию инновационных технологий в области компьютерного зрения для анализа медицинских изображений и дальнейшего применения этих технологий в системе здравоохранения"*, проводимого Центром диагностики и телемедицины Департамента здравоохранения Москвы.

## Продукты

- Сервисы искусственного интеллекта (ИИ-сервисы);
- Разметчик изображений для формирования датасетов для обучения нейронных сетей;
- Программный комплекс для автоматизации процесса разработки ИИ-сервисов.

## Рабочие задачи

Документированием занимались мы с моей старшей коллегой. Для выполнения административных процедур компания привлекала внешних консультантов, имеющих релевантный опыт.

Документы, которые мы составляли, в основном предназначались:

- для регистрации разработок в реестре отечественного ПО Минцифры, чтобы получать гранты и преференции от государства;
- для регистрации в реестре медицинских изделий Росздравнадзора, чтобы применять наши разработки в клинической практике;
- для регистрации в реестре программ для ЭВМ и БД Роспатента, чтобы защитить нашу интеллектуальную собственность;
- для участия в грантовых конкурсах;
- для участия в системе госзакупок;
- для внутренних нужд и процессов компании.
