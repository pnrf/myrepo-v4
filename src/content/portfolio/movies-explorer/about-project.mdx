# Описание проекта

В 2022 году я прошёл обучение в Яндекс Практикуме на курсе ["Веб-разработчик"](https://practicum.yandex.ru/web/). Изучил базовые подходы и технологии fullstack-разработки, чтобы понимать, как создаются программные продукты.

В качестве дипломной работы разработал веб-приложение «Movies Explorer» с личным кабинетом для просмотра каталога фильмов. Компактный аналог «Кинопоиска».

Спустя 2,5 года многие технологии, которым нас обучали, основательно забылись. Чтобы освежить знания и пополнить портфолио, написал документацию к своей дипломной работе.

## Ссылки на проект

В 2022 году мой дипломный проект был размещен в Яндекс Облаке и доступен по публичному IP-адресу. В настоящий момент проект доступен только для просмотра программного кода в Github-репозитории и для запуска его на локальном сервере.
