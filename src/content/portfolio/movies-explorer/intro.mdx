# Описание веб-приложения

Movies Explorer – веб-приложение с открытым программным кодом. Оно позволяет пользователям:

- посмотреть каталог фильмов;
- найти фильм по запросу или по фильтрам;
- зарегистрироваться и войти в личный кабинет;
- добавить фильм в список избранных;
- посмотреть список избранных фильмов.

## Структура приложения

Веб-приложение состоит из нескольких страниц:

- Главная страница. Содержит информацию с описанием приложения.
- Страница с каталогом фильмов.
- Страница с избранными фильмами.
- Страница регистрации нового пользователя.
- Страница авторизации зарегистрированного пользователя.
- Страница редактирования профиля пользователя.
