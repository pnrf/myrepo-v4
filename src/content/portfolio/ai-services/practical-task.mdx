# Практическая задача

## Назначение сервиса

«RadiMedAI Protocol» предназначен для автоматизации описания и интерпретации рентгенологических исследований в системе поддержки принятия врачебных решений (СППВР).

Использование сервиса направлено на:
- оптимизацию медицинской помощи, 
- снижение нагрузки на врачей-рентгенологов, 
- повышение эффективности их работы,
- снижение риска врачебной ошибки.

## Принцип работы

**Алгоритм сервиса:**
1. анализирует DICOM-изображения компьютерной томографии органов малого таза с контрастированием;
2. выявляет признаки рака предстательной железы;
3. формирует серию изображений с выделением патологических областей;
4. предоставляет врачу предзаполненный протокол исследования.

**Врач-рентгенолог:**
1. изучает выделенные области;
2. завершает формирование протокола, принимая или корректируя рекомендации ИИ-сервиса на основе своего профессионального суждения.

## Основные задачи
* Обеспечение информационного обмена между системой архива медицинских изображений (PACS) и ИИ-сервисом.
* Сокращение времени обработки диагностического исследования.
* Уменьшение количества ошибок, связанных с ручной обработкой диагностических исследований (медицинских изображений).
* Автоматизация процессов обмена информацией при проведении диагностических исследований для интерпретации результатов компьютерной томографии органов малого таза с внутривенным контрастированием при выявлении признаков рака предстательной железы.

## Результаты работы
* Текстовое заключение в формате DICOM SR, на основании которого возможно формирование электронной медицинской записи (предзаполненный протокол диагностического исследования).
* Дополнительная серия изображений, содержащая результаты обработки исследований ИИ-сервисом (с контурированием находок и/или наложением масок).
