# Описание проекта

В 2023–2024 годах я работал в компании, которая разрабатывала программные решения для медицинской отрасли:

- **ИИ-сервисы** для диагностики патологий на медицинских снимках (маммография, рентгенография, КТ/НДКТ) по различным нозологиям;

- **разметчик изображений** для создания верифицированных наборов данных (дата-сетов), используемых в обучении нейронных сетей;

- **программный комплекс** для автоматизации процессов разработки ИИ-сервисов и их интеграции в медицинскую информационную систему.

Компания принимала активное участие в [государственном эксперименте](https://mosmed.ai/ai/), проводимом [Центром диагностики и телемедицины](https://telemedai.ru/) Департамента здравоохранения города Москвы.

## Задачи

1. Разработка документов для регистрации наших продуктов: 
	- в реестре отечественного ПО Минцифры;
	- в реестре медицинских изделий Росздравнадзора;
	- в реестре программ для ЭВМ и БД Роспатента.
2. Разработка документов для:
	- участия в грантовых конкурсах;
	- участия в системе госзакупок.

## Особенности

ИИ-сервисы подпадали сразу под 3 категории:

1. медицинское изделие, подлежащее государственной регистрации;
2. программное обеспечение, входящее в состав автоматизированной информационной системы;
3. сервис искусственного интеллекта в медицине.

Поэтому всю техническую документацию мы разрабатывали с учетом ГОСТ 19, 34, 59 серий и системы менеджмента качества. Нормоконтолем и оформлением документов занимались внешние консультанты, которых компания привлекала для выполнения административных процедур.
