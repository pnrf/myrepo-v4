# Описание проекта

*В 2023–2024 годах я работал в компании, которая разрабатывала программные решения для медицинской отрасли. Компания принимала активное участие в [государственном эксперименте](https://mosmed.ai/ai/), проводимом Центром диагностики и телемедицины Департамента здравоохранения города Москвы.*

## Цель проекта

Коммерциализировать программные продукты компании.

## Задачи

1. Разработать документы для регистрации программных продуктов: 
	- в реестре отечественного ПО Минцифры;
	- в реестре медицинских изделий Росздравнадзора;
	- в реестре программ для ЭВМ и БД Роспатента.
2. Разработать документы для участия:
	- в грантовых конкурсах;
	- в системе госзакупок.

## Продукты

- **ИИ-сервисы** для диагностики патологий на медицинских снимках (маммография, рентгенография, КТ/НДКТ) по различным нозологиям;
- **разметчик изображений** для создания верифицированных наборов медицинских данных (дата-сетов), используемых в обучении нейронных сетей;
- **программный комплекс** для автоматизации процессов разработки ИИ-сервисов и их интеграции в медицинскую информационную систему.

## Особенности

Наши ИИ-сервисы подпадали сразу под 3 категории:

1. медицинское изделие, подлежащее государственной регистрации;
2. программное обеспечение, входящее в состав автоматизированной информационной системы;
3. сервис искусственного интеллекта в медицине.

Поэтому техническую документацию мы разрабатывали с учетом ГОСТ 2, 19, 34, 59 серий и системы менеджмента качества. Нормоконтролем и оформлением документов занимались внешние консультанты, которых компания привлекала для выполнения административных процедур.

## Результаты

На момент моего увольнения из компании достигнуты следующие результаты.

**ИИ-сервисы:**
- Один из девяти сервисов внесен в реестр отечественного ПО Минцифры.
- Три из девяти сервисов зарегистрированы в реестре программ для ЭВМ и БД Роспатента.
- Три сервиса успешно прошли предварительные клинико-технические испытания. Велась активная работа по их регистрации в Росздравнадзоре (реестр медицинских изделий).

**Разметчик изображений:**
- Успешно апробирован врачами. Активно применяется в работе компании.

**Программный комплекс:**
- Составлены техническое задание на разработку Программного комплекса и документация для участия в системе госзакупок. MVP перешел в стадию тестирования.
- Пройден отборочный этап на грантовом техноконкурсе UP Great – I'm Doctor.
