# Описание ИИ-сервиса

**ИИ-сервис:** "RadiMedAI Protocol"

**Краткое описание:** Сервис искусственного интеллекта по интерпретации изображений компьютерной томографии органов малого таза с внутривенным контрастированием для выявления признаков рака предстательной железы.

**Заказчик разработки:** ФГБУ «Лечебный центр»

**Разработчик:** ООО «РадиМедИИ»
