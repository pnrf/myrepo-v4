## О документе

Наша компания [разрабатывала сервисы искусственного интеллекта](/portfolio/projects/razrabotka-ii-servisov) для лучевой диагностики. Методом компьютерного зрения нейронные сети распознавали признаки патологий на медицинских снимках и формировали протокол рентгенологического исследования.

Для применения ИИ-сервисов в клинической практике их требуется регистрировать в качестве медицинских изделий. Руководство по эксплуатации программного обеспечения является обязательным для подачи в Росздравнадзор (в числе прочих документов).

## Особенности

Наши ИИ-сервисы подпадали сразу под 3 категории:

1. медицинское изделие, подлежащее государственной регистрации;
2. программное обеспечение, входящее в состав автоматизированной медицинской информационной системы;
3. сервис искусственного интеллекта в медицине.

Поэтому всю техническую документацию нам потребовалось разрабатывать с учетом ГОСТ 2, 19, 34 и 59 серий.
