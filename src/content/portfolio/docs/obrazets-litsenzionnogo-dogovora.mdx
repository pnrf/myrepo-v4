# Образец лицензионного договора

<Steps>

### О документе

Лицензионный договор предусмотрен статьей 1235 Гражданского кодекса. Этот документ служит правовым основанием для передачи результатов интеллектуальной деятельности (например, экземпляра программы для ЭВМ) во временное пользование другой стороне (лицензиату, пользователю), а также для начисления и получения лицензионного вознаграждения.

Закон предъявляет особые требования к содержанию лицензионного договора. Не выполнение этих требований влечет недействительность сделки и прочие негативные последствия.

### Исполнение

Наша компания [регистрировала свои разработки](../projects/registratsiya-rossiyskogo-po-v-reyestre-mintsifry.mdx) (ИИ-сервисы) в реестре российского ПО. Мне поручили составить образец лицензионного договора, поскольку он входит в комплект документов, подаваемых в Минцифры.
