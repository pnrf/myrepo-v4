# Описание проекта

*В 2025 году возникла необходимость в поиске работы. Для того, чтобы общаться с рекрутерами и тимлидами предметно, сделал сайт-портфолио.*

## Цель проекта

Разработать сайт-портфолио для поиска работы.

## Задачи проекта

- Выбрать и изучить технологический стек.
- Разработать дизайн, структуру и содержание сайта.
- Подготовить контент для публикации на сайте.
- Сверстать сайт, разместить его на домене.

## Результаты

Сайт доступен по адресу: [https://www.myrepo.ru](https://www.myrepo.ru)

## План развития

* Подключение аналитических инструментов.
* Расширение контента портфолио, базы знаний и блога.
* Миграция с Nextra на Docusaurus.
