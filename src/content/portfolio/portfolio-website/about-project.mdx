# Описание проекта

## Введение

**Сайт-портфолио "Мой репозиторий"** – это мой личный сайт для поиска работы. "Песочница" для развития моих навыков программирования, составления текстовых описаний и графических схем. 

Проект содержит [портфолио с документацией](/portfolio), [базу знаний](/knowledge-base) и [блог](/blog). 

Сайт доступен для широкой аудитории по адресу: [Myrepo.ru](https://www.myrepo.ru).

## Особенности проекта
* Проект не использует сервисы Яндекса и Google (пока)
* Отсутствуют рекомендательные технологии
* Имеет рабочую живую версию по адресу: [myrepo.ru](https://www.myrepo.ru)

## Потенциал развития
Проект может быть расширен путем добавления:
* Аналитических инструментов
* Интеграций с популярными сервисами
* Дополнительных разделов портфолио, базы знаний и блога
