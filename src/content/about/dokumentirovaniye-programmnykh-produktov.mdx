# Документирование программных продуктов


## Какие документы разрабатываю

По ГОСТ 19.101-77 - ЕСПД. Виды программ и программных документов:

- Спецификация (ГОСТ 19.202-78)
- Ведомость держателей подлинников (ГОСТ 19.403-79)
- Текст программы (ГОСТ 19.401-78)
- Описание программы (ГОСТ 19.402-78)
- Программа и методика испытаний (ГОСТ 19.301-79)
- Техническое задание (ГОСТ 19.201-78)
- Пояснительная записка (ГОСТ 19.404-79)
- Эксплуатационная документация:
	- Ведомость Эксплуатационных документов (ГОСТ 19.507-79)
	- Формуляр (ГОСТ 19.501-78)
	- Описание применения (ГОСТ 19.502-78)
	- Руководство системного программиста (ГОСТ 19.503-79)
	- Руководство программиста (ГОСТ 19.504-79)
	- Руководство пользователя (оператора) (ГОСТ 19.505-79)
- Описание языка (ГОСТ 19.506-79)
- Руководство по техническому обслуживанию (ГОСТ 19.508-79)
- Схемы алгоритмов, программ, данных и систем (ГОСТ 19.701-90)
